# Checklist de Onboarding (Restaurante/Colmado)

## 1) Datos del negocio
- [ ] Nombre comercial confirmado
- [ ] Tipo de negocio definido: `restaurant` o `colmado`
- [ ] Telefono principal y WhatsApp verificados
- [ ] Direccion completa validada
- [ ] Coordenadas exactas (Google Maps) guardadas
- [ ] Logo/foto principal recibida (opcional)

## 2) Configuracion comercial
- [ ] Comision 8% explicada y aceptada
- [ ] Trial de 3 meses + suscripcion RD$1,500/mes explicado
- [ ] Periodo de gracia de 14 dias explicado
- [ ] Firma de acuerdo comercial completada

## 3) Configuracion tecnica
- [ ] Negocio creado en panel admin
- [ ] PIN de merchant generado y entregado de forma segura
- [ ] Acceso a panel merchant probado (login correcto)
- [ ] Estado de suscripcion verificado (`trial`)

## 4) Carga de menu/productos
- [ ] Minimo 10 productos cargados (nombre + precio)
- [ ] Categorias asignadas correctamente
- [ ] Productos agotados marcados como no disponibles
- [ ] Precios confirmados por el dueno/encargado

## 5) Flujo operativo
- [ ] Se hizo 1 pedido demo desde app movil
- [ ] Comercio actualizo estados del pedido en panel
- [ ] Pedido llego a estado `delivered`
- [ ] Comision semanal reflejada en settlement

## 6) Entrenamiento corto (15-20 min)
- [ ] Como aceptar/preparar/cerrar pedidos
- [ ] Que hacer ante cancelaciones
- [ ] Buenas practicas de tiempo de respuesta
- [ ] Proceso de cobro semanal explicado

## 7) Salida en vivo
- [ ] Fecha/hora de activacion acordada
- [ ] Responsable interno designado (turno principal)
- [ ] Canal de soporte definido (WhatsApp/llamada)
- [ ] Seguimiento de primera semana agendado
